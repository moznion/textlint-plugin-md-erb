textlint-plugin-md-erb
==

A textlint plugin to support the erb file with markdown format (.md.erb).

Install
--

```
npm install textlint-plugin-md-erb
```

Usage
--

Add plugin into your `.textlintrc`;

```
{
  "plugins": [
    "md-erb"
  ]
}
```

And run the textlint.

Build
--

```
npm install
npm run-script build
```
